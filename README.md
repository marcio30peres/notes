# Notes
Este repositório não conterá projetos, mas sim anotações sobre o aprendizado da biblioteca React.

## Motivação
Sempre fui apaixonado por tecnologia, mas há aproximadamente dois anos comecei a me comprometer com o aprendizado de programação vislumbrando esta como sendo uma real possibilidade para meu futuro profissional. Assim iniciei meus estudos no curso técnico em desenvolvimento de sistemas que estou concluindo agora em 2021.

Como o aprendizado nunca pode parar e sempre haverá uma busca por me especializar e capacitar como desenvolvedor web front-end, decidi organizar esse repositório como uma alternativa para facilitar futuras revisões e também para contribuir com outros desenvolvedores interessados no assunto.

## Conteúdos

### Porque arquivos com extensão .md?
Esta extensão foi escolhida por dois motivos principais:
* Permite mesclar a explicação em texto com realce de sintaxe nos códigos, assim facilitando a adição de exemplos para melhor compreensão durante a conculta do material
* Esta visualização pode ser feita sem sair do seu browser já que o Github suporta arquivos com essa extensão.

### Organização
Os conteúdos estão organizados por arquivos que agregam conteúdos relacionados. Tentarei ao máximo criar títulos que realmente traduzam o conteúdo veiculado facilitando assim a consulta. 

Dentro de cada arquivo haverá um título principal referente ao tema global das anotações. Na sequência os títulos secundários servirão para referenciar as principais etapas para construção dos exemplos ou fundamentação das teorias. Já os títulos terciários serão os passos práticos mais específicos, exemplos de código e aplicações. Por fim, as observações e comentários servirão para comunicar curiosidades e informações interessantes para o desenvolvedor, relacionando os trechos de código com as explicações prévias.

## Próximos passos
Este é apenas o começo desse projeto. Pretendo seguir alimentando este repositório com conteúdos não apenas sobre React, mas também outros frameworks e bibliotecas. Espero que esse conhecimento seja valorizado e chegue àqueles que tem o potencial e motivação para mudar o mundo com ideias inovadoras, soluções criativas e se colocando em movimento para transformar a realidade local, do nosso país e além.
